# MarkZero Capacidades y Caracteristicas

Este documento detalla las funciones actuales que el motor de MarkZero es capaz de procesar y renderizar.

## Sintaxis de Texto
- Encabezados: Soporte desde H1 hasta H6 usando el simbolo #.
- Enfatizado: Negrita (** o __) y Cursiva (* o _).
- Tachado: Uso de doble virgulilla (~~) para tachar texto.
- Codigo en linea: Uso de comillas simples invertidas para fragmentos cortos.
- Parrafos: El motor envuelve automaticamente el texto suelto en etiquetas p.

## Listas
- Listas Desordenadas: Uso de guiones (-) con soporte para multiples niveles de sangria.
- Listas Ordenadas: Uso de numeros (1. 2.) con soporte para anidacion.
- Tareas: Checkboxes funcionales marcados con [ ] o [x].

## Bloques de Contenido
- Citas: Soporte para blockquotes simples y anidados usando el simbolo >.
- Tablas: Renderizado de tablas completas con alineacion y separadores.
- Codigo: Bloques de codigo multilinea con resaltado visual y boton de copiado integrado.
- Separadores: Lineas horizontales usando tres guiones (---).

## Multimedia y Enlaces
- Enlaces: Formato estandar [texto](url) y enlaces automaticos.
- Imagenes: Insercion de imagenes con texto alternativo y rutas.
- Notas al pie: Soporte para referencias y sus definiciones al estilo [^1].

## Funciones del Sistema
- Modo Desarrollo: Servidor local con Live Reload (recarga automatica al guardar).
- Build Limpio: Generacion de archivos HTML estaticos sin scripts de desarrollo.
- Plantillas: Estructura base modular para cabeceras y pies de pagina.
- Estilo: Tema oscuro predefinido basado en la estetica de GitHub.
